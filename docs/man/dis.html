DIS - Display single image using ImageMagick

(4/16/16)

PURPOSE

Display single image using ImageMagick. Requires that your system has Imagemagick installed and available as: 'convert'. Assumes that SPIDER images are 'Most significant byte first' which is usually true and that the images may have negative intensities.   Example.

SEE ALSO

CP TO JPG [Copy SPIDER file to a JPEG format]

USAGE

.OPERATION: DIS

.INPUT FILE: PIC001
[Enter the name of the image file. Does not work with volumes or bare stacks. Will automatically create a JPEG copy of the image with the same filename but having '.jpg' extension.]

.IMAGEMAGICK DISPLAY OPTIONS (or <CR>): -resize 50%
[Enter any options for the Imagemagick display command.]
