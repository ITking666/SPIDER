INSTALLATION AND RELEASE NOTES

  • INSTALLATION     The distribution consists of three main modules both of which contain all the relevant source code (most of which is released under the GPLv2 License) as well as some precompiled executables .

    SPIDER
    The SPIDER module is used for computations that do not require graphical interaction or visualization. The distribution contains precompiled executables for:
    • Linux on AMD 32 and 64
    • Linux on Intel 32 and 64
    • OS X on Intel 32 and 64

    SPIDER is written in FORTRAN 2003 and "Makefiles" are provided to build SPIDER on Linux, IBM AIX, OSX, and some legacy systems. These "Makefiles" will have have to be modified depending on your choice of compiler, location of libraries, etc.

    Installing SPIDER On:   Linux & OS X

    Web
    The Web module is used for operations that involve graphical interaction with the data such as display of image files. It is a point-and-click interface that uses either the MOTIF/X-WINDOW system or Java (JWeb). The distribution contains precompiled MOTIF/X-WINDOW executables for:
    • Linux on AMD & Intel
    • OS X on Intel

    The MOTIF/X-WINDOW version of Web is written in C and "Makefiles" are provided to build Web on Linux, OSX, and some legacy systems. These "Makefiles" will have have to be modified depending on your choice of compiler, location of libraries, etc.

    Installing Web On:   Linux     OS X

    The Java version of Web can be used on Linux on Intel & AMD, as well as Microsoft Windows. The distribution includes: Java Run Time Environment, Java Advanced Imaging Run Time Environment, and Jweb code.

    Installing JWeb On;   Linux     Windows
