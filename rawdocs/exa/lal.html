Usage Example - Operation: 'LA L'

Put text label inside an image

Procedure used: lal.spi

.OPERATION: LA L ;   Put label inside an image
.INPUT FILE: sav_pp ;   Input file name
.OUTPUT FILE: lal ;   New file name
.LABEL: LA_L_001 ;   Label
.LOCATION: 10, 30 ;   Label
.LABEL: ;   Stop labeling
