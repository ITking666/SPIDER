Usage Example - Operation: 'CP TO XPLOR'

Copy SPIDER file To XPLOR image

Procedure used: cptoxplor.spi

.OPERATION: CP TO XPLOR ;   Copy SPIDER image To XPLOR
.SPIDER INPUT FILE: sav_mo_sine ;   SPIDER image (input)
.PIXEL SIZE: 4 ;   Angstroms / pixel
.XPLOR OUTPUT FILE: sav_cptoxplor ;   XPLOR file (output)


INPUT IMAGE
sav_mo_sine


XPLOR OUTPUT
sav_cptoxplor
