Usage Example - Operation: 'AP SHC'

Alignment - multi-reference, fast exhaustive rotation & shift

Procedure used: apshc.spi

.OPERATION: AP SHC ;   Alignment determination
.TEMPLATE FOR REFERENCE IMAGES: sav_nat_proj_83@**;   Ref image name template (input)
.FILE NUMBERS OR SELECTION DOC. FILE NAME: 1-83 ;   List of reference images
.TRANSLATION SEARCH RANGE IN X, IN Y (OPTIONAL), STEP SIZE: 5,5,1 ;   Translation shifts, shift step
.FIRST, LAST RING, RING STEP, & RAY STEP: 5,58,1,1 ;   First, last ring, ring skip, ray skip
.REFERENCE IMAGES ANGLES DOCUMENT FILE: sav_voea_doc_83 ;   Reference angles (input)
.TEMPLATE FOR IMAGE SERIES TO BE ALIGNED: sav_nat_part@** ;   Exp image name template (input)
.FILE NUMBERS OR SELECTION DOC. FILE NAME: 1-5 ;   List of Exp images
.EXPERIMENTAL IMAGES ALIGNMENT DOCUMENT FILE: * ;   No exp image angles doc file (input)
.RANGE OF PROJECTION ANGLE SEARCH & ANGLE CHANGE THRESHOLD: 0,0 ;   No angular restriction
.CHECK MIRRORED POSITIONS?, SHIFT AND ROTATE INPUT?, DENOISE (Y/N):Y,N,N ;   Check mirrors, no RTSQ first, No denoise
.OUTPUT ALIGNMENT DOCUMENT FILE: apshc_ali_doc ;   Alignment doc file (output)
