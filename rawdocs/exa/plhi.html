Usage Example - Operation: 'PL HI'

Plot Doc file data as histogram into Postscript file

Procedure used: plhi.spi

.OPERATION: PL HI ;   Plot Histogram - from Doc file
.DOCUMENT FILE: savsortdoc ;   Doc fIle (input)
.FIRST AND LAST KEY NUMBERS: 1, 112 ;   Plot reg. 2 on axis
.REGISTER (COLUMN) IN DOC FILE USED FOR HISTOGRAM: 2 ;   Histogram of register = 2
.STARTING X VALUE, INTERVAL: 1, 50 ;   Histogram start, bin size
.ENDING X VALUE: 250 ;   Histogram end
.POSTSCRIPT OUTPUT FILE: plhi ;   Postscript plot (output)
.NEW LOWER, UPPER AXIS BOUNDS OR <CR>: 0,20 ;   Y axis
.NEW AXIS LABEL UNIT AND TICKS / LABEL OR <CR>: 5, 5 ;   Label unit & tick unit
.LABEL NO. TO EDIT... OR <CR> TO CONTINUE: ;   <CR> = Continue
.NEW AXIS LABEL UNIT AND TICKS / LABEL OR <CR>: 0, 250 ;   X axis
.NEW LOWER, UPPER AXIS BOUNDS OR <CR>>: 50,10 ;   Label unit & tick unit
.LABEL NO. TO EDIT... OR <CR> TO CONTINUE: ;   <CR> = Continue


INPUT DOC FILE
savsortdoc0

OUTPUT PLOT
plhi_plot.ps
