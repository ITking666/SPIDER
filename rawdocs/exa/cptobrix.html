Usage Example - Operation: 'CP TO BRIX'

Copy SPIDER file To BRIX image

Procedure used: cptobrix.spi

.OPERATION: CP TO BRIX ;   Copy SPIDER image To BRIX
.SPIDER INPUT FILE: savpad_nik_sli_45 ;   SPIDER image (input)
.BRIX OUTPUT FILE: cptobrix.brix ;   BRIX file (output)
.SAMPLING DISTANCE: 4 ;   Sampling distance, Angstroms / pixel
.DEFAULT UNIT CELL IS NX,NY,NZ. OVERWRITE (Y/N)?: No ;   Do not change cell


INPUT IMAGE
savpad_nik_sli_45


BRIX OUTPUT IMAGE
NO VIEWER AVAILABLE
cptobrix.brix
