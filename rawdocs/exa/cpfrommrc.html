Usage Example - Operation: 'CP FROM MRC'

Create SPIDER image/volume from MRC (CCP4) file

Procedure used: cpfrommrc.spi

.OPERATION: CP FROM MRC ;   Create SPIDER image from MRC file
.MRC (CCP4) INPUT FILE: sav_img_mrc.mrc ;   MRC image (input)
.SPIDER OUTPUT FILE: cpfrommrc_img ;   SPIDER image (output)
.FLIP BYTE ORDERING? (Y/N), FLIP TOP/BOTTEM? (Y/N): NO,YES ;   Don't flip byte order, Invert top/bottem


OUTPUT IMAGE
cpfrommrc_img
