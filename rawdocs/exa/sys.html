Usage Example - Operation: 'SYS'

Operating system call

Procedure used: ./images/sys.spi

.OPERATION: FI H [nx],[max] ;   Get file information
.INPUT FILE: sav_sine_wave ;   Image (input)
.HEADER VARIABLE(S): NX, MAX ;   X dimension, max. intensity
.  
.OPERATION: SYS ;   Put register variable substituted msg. in results
.SYSTEM COMMAND: echo " Columns: {***[nx]}"
.  
.OPERATION: SYS ;   Put register variable substituted msg. in results
.SYSTEM COMMAND: echo " Columns: {%I3%[nx]}"
.  
.OPERATION: SYS ;   Put register variable substituted msg. in results
.SYSTEM COMMAND: echo " Maximum: {%f5.2%[max]}"
.  
.OPERATION: SYS ;   Put string variable substituted msg. in results
.SYSTEM COMMAND: ls -l [in].$DATEXT
.  
.OPERATION: [n]=2 ;   Set register variable
.OPERATION: SYS ;   Rename/renumber file
.SYSTEM COMMAND: cp sav_sin_wave.dat jnk{***[n]}.dat
.
