Usage Example - Operation: 'CP to MRC'

Copy SPIDER file To MRC file

Procedure used: cptomrc.spi

.OPERATION: CP TO MRC ;   Copy SPIDER file To MRC file
.SPIDER INPUT FILE: savpad_nik_sli_45 ;   SPIDER image (input)
.MRC OUTPUT FILE: cptomrc ;   MRC image (output)
.MRC DATA LENGTH (8/32 BITS), FLIP TOP/BOTTOM =1 (0/1): 32, 1 ;   32 bit output, Bottem at top
.STARTING X, Y, & Z OF MRC IMAGE: ;   Use default start
.ANGSTROMS/PIXEL FOR X, Y, & Z AXIS: 12, 12, 12 ;   Pixel size
.X, Y, & Z ORIGIN OF MRC IMAGE: ;   Use default origin
. ;  
.OPERATION: CP FROM MRC ;   Create SPIDER image from MRC file
.MRC (CCP4) INPUT FILE: cptomrc ;   MRC image (input)
.SPIDER OUTPUT FILE: cptofrommrc ;   SPIDER image (output)
.FLIP BYTE ORDERING? (Y/N), FLIP TOP/BOTTEM? (Y/N): NO,YES ;   Don't flip byte order, Invert top/bottem


INPUT IMAGE
savpad_nik_sli_45
OUTPUT IMAGE
cptofrommrc
RESULTS FILE
