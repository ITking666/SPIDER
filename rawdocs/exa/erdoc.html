Usage Example - Operation: 'ER DOC'

Erode a binary image/volume and save ultimate eroded pixels in doc file

Procedure used: erdoc.spi

.OPERATION: ER DOC ;   Erode a binary image and save UEP
.INPUT FILE: sav_bin ;   File name (input)
.OUTPUT FILE: erdoc ;   File name (output)
.BOX OR CROSS NEIGHBORHOOD: Box ;   Box shaped neighborhood
.LENGTH OF NEIGHBORHOOD: 3 ;   Neighborhood length
.ERODE IF NUMBER OF OCCUPIED NEIGHBORS IS < THAN: 6 ;   Erosion neighbors
.DOCUMENT FILE: erdoc_doc ;   Doc file (output)


INPUT IMAGE OUTPUT IMAGE
sav_bin erdoc
OUTPUT DOC FILE
erdoc_doc
