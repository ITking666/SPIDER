Usage Example - Operation: 'RB 32F '

Rotate, Shift, Sampled, Back Projection in Fourier space

Procedure used: rb32f.spi

.OPERATION: RB 32F ;   Reconstruct volume from projections
.TEMPLATE FOR IMAGE FILES: sav_data ;   Unaligned projections (input)
.FILE NUMBERS OR SELECTION DOC. FILE: 1-12 ;   List of projections (input)
.ANGLES DOC FILE: sav_align_data_doc ;   Projection angles doc file (input)
.SYMMETRIES DOC FILE: * ;   No symmetries
.TRANSFORMED OUTPUT IMAGES TEMPLATE: * ;   No rotated image files (output)
.RECONSTRUCTED VOLUME: rb32f ;   Reconstructed volume (output)
.FIRST SAMPLE VOLUME: rb32f_sub1 ;   First reconstructed volume (output)
.SECOND SAMPLE VOLUME: rb32f_sub2 ;   Second reconstructed volume (output)
