Usage Example - Operation: 'PK 3R'

Search for peak(s) in a SPIDER volume over restricted neighborhood

Procedure used: pk3r.spi

.OPERATION: PK 3 [x],[y],[z],[xp],[yp],[zp],[pk] ;   Search for 4 highest peaks in image
.INPUT FILE: sav_crc_vol ;   Image (input)
.MAXIMA(+) OR MINIMA(-): + ;   Find maxima
.NUMBER OF PEAKS, CENTER ORIGIN OVERRIDE (1/0): 5, 1 ;   Number of peaks, relative to center
.X, Y, & Z RADII OF EXCLUDED NEIGHBORHOOD: 6, 6, 3 ;   Exluded neighborhood elipse
.X, Y, & Z ORIGIN COORDINATES: 1, 1, , 1 ;   Origin
.PEAK NUMBER FOR RATIO: 1 ;   Compare to peak 1
.BOX SELECTION (Y/N): No ;   No box selection
.DOCUMENT FILE: pk3r_doc ;   No box selection


INPUT VOLUME
(slices)
sav_crc_vol


RESULTS FILE
pk3d_resu


OUTPUT DOC FILE
pk3r_doc
